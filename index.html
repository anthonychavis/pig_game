Rules

Objective: Be the first player to score 💯❗

• To start the game, player 1 pushes the Roll Die button.

• The player who's turn it is can roll the die as many times as desired, as long as it is still that player's turn. • However if the value of the die is 1, any value in the Current Score is lost and it becomes the next player's turn. • If the die is not 1, the value is added to the player's Current Score.

• The Hold button is used to move the value of the Current Score to the player's main score. • When the Hold button is used, the value of the Current Score is moved to the player's main score. • It is then the next player's turn to roll the die.

• If you're feeling lucky, keep rolling to increase your Current Score; just remember to Hold before that unlucky number 1️⃣ pops up❗

• Once the value of a player's main score is at least 100, that player is the winner of the game! 🎉

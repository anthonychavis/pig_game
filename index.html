Rules

Objective: Be the first player to score 100!
• To start the game, player 1 pushes the Roll Die button.
• If the die is not 1, the value is added to the player's Current Score. If the value is 1, any value in the Current Score is lost and it becomes the next player's turn.
• When the Hold button is used, the value of the Current Score is moved to the player's main score. It is then the next player's turn to roll the die.
• Once the value of a player's main score is at least 100, that player is the winner of the game! 🎉
